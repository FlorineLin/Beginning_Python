Creating and Using Dictionaries¶

In [7]:
phonebook={'Alice';'2341','Beth':'9102','Cecil';'3258'}
In [8]:
phonebook['Cecil']
Out[8]:
'3258'

字典由键及其对应值的键值对（称为条目）组成。在这个例子中，姓名是键，电话号码是值。每个键与其值之间用冒号（:）分隔，条目之间用逗号分隔，整个字典用花括号括起来。空字典（没有任何条目的字典）只用两个花括号表示，如下所示：{}。

# A simple database
# A dictionary with person names as keys. Each person is represented as
# another dictionary with the keys 'phone' and 'addr' referring to their phone
# number and address, respectively.
people={
    'Alice': {
        'phone':'2341',
        'addr':'Foo drive 23'
    },
    'Beth': {
        'phone':'9102',
        'addr':'Bar street 42'
    },
    'Cecil': {
        'phone':'3158',
        'addr':'Baz avenue 90'
    }
}
# Descriptive labels for the phone number and address. These will be used
# when printing the output.
labels={
    'phone':'phone number',
    'addr':'address'
}
name=input('Name: ')
# Are we looking for a phone number or an address?
request=input('Phone number (p) or address (a)?')
# Use the correct key:
if request == 'p':key='phone' 
if request == 'a':key='addr'
# 两个等号 == 是因为它代表比较运算，用于判断两边的值是否相等。
# 而单个等号 = 是赋值运算，用于将右边的值赋给左边的变量
# Only try to print information iif the name is avalid key in 
# our dictionary:
if name in people:print("{}'s {} is {}.".format(name, labels[key],people[name][key]))

Beth's phone number is 9102.

copy¶

In [58]:
x={'username':'admin','machines':['foo','bar','baz']}
y=x.copy()
y['username']='mlh'
y['machines'].remove('bar')
y
Out[58]:
{'username': 'mlh', 'machines': ['foo', 'baz']}
In [59]:
x
Out[59]:
{'username': 'admin', 'machines': ['foo', 'baz']}
如你所见，当你替换副本中的某个值时，原始数据不会受到影响。然而，如果你修改某个值（在原地修改，而非替换它），原始数据也会发生变化，因为两者存储的是同一个值（就像本例中的 “machines” 列表）。避免这个问题的一种方法是进行深拷贝，即同时拷贝值本身以及它们所包含的任何值，依此类推。你可以使用 copy 模块中的 deepcopy 函数来实现这一点。
In [62]:
from copy import deepcopy
d={} #  表示创建一个空字典，变量名为 d
d['names']=['Alfred','Bertrand'] # 给字典 d 添加一个键为 'names' 的条目，对应的值是一个包含两个字符串元素的列表
c=d.copy()
dc=deepcopy(d)
d['names'].append('Clibe')
c
Out[62]:
{'names': ['Alfred', 'Bertrand', 'Clibe']}
In [63]:
dc
Out[63]:
{'names': ['Alfred', 'Bertrand']}

popitem¶

popitem 方法类似于 list.pop，后者会弹出列表的最后一个元素。然而，与 list.pop 不同的是，popitem 会弹出任意一个元素，因为字典没有 “最后一个元素”，也根本没有任何顺序可言。如果你想以高效的方式逐个移除并处理元素（无需先获取键的列表），这可能会非常有用。

In [94]:
d={'url':'https://www.python.org','spam':0,'title':'Python Web Site'}
d.popitem()
Out[94]:
('title', 'Python Web Site')
In [95]:
d
Out[95]:
{'url': 'https://www.python.org', 'spam': 0}
